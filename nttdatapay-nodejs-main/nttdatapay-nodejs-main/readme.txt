1. This is demo project for Node Js AIPAY tested upto Node version 20.12.1.
2. Install all the dependencies for this project with the command npm install.
3. App.js is a main file in which we have mentioned all logic required for AIPAY implementation.
4. you have to run the application first by using node app.js command.
5. Once peoject runs with with port 3000 we need to open link manually in web browser like http://localhost:3000.
6. On main page you can see sample merchant shop with AtomTokenId, TransactionId and amount.
7. Once you get token Id from app.js you will get above index.html view file based on API results.
8. You have to change CDN link mentioned inside index.html file according to UAT/Production.
